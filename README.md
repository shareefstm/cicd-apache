# CI/CD Apache Deployment using Terraform and Jenkins

## Overview
This project automates the deployment of an Apache web server on AWS using Terraform and Jenkins.

## Files
- apache.tf - Apache EC2 infrastructure
- cicd.tf - Jenkins CI/CD infrastructure
- apache.script.sh - Apache installation script
- jenkins-cicd.script.sh - Jenkins installation script

## Prerequisites
- AWS Account
- Terraform
- Git
- Jenkins

## Deployment

Initialize Terraform:

terraform init

Validate:

terraform validate

Plan:

terraform plan

Apply:

terraform apply
